# Vuetify-project

#### Install Vue CLI 3
create folder 'vuetify-learning'
```
npm install @vue/cli -g
> default (babel, eslint)
```

#### Create 'my-app' project
```
vue create my-app
```

#### Install Vuetify into the project
```
cd my-app
```
```
vue add vuetify
> default (Recommended) 
```

#### Run server
```
npm run serve
http://localhost:8080/
```

#### Referance
https://vuetifyjs.com/en/getting-started/quick-start
